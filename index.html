How Auto-Crawling Works
GitHub Actions every ~2.5 hours
▼
⏰
Every ~2.5 hours automatically — GitHub Actions runs crawler.py. Crawls Reddit (direct API), Google News RSS, and Google-indexed Quora, Medium, YouTube, Shiksha, CollegeDunia, LinkedIn, Twitter and more.
📦
Data saved as JSON files in your repo under data/ folder. This dashboard fetches those files directly from GitHub raw URLs.
🔄
Dashboard auto-refreshes every 5 minutes. The crawl status dot turns green (fresh), amber (3-8h old), red (stale).
🔑
API keys are optional — crawler works free without any keys. Keys just improve data quality and volume.
📋
My Content tabs show YOUR manually logged content from Google Sheets — Reddit posts you wrote, Quora answers you seeded, etc.
📡
